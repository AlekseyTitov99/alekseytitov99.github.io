Data Structures

There are a lot of data structures that can complete a project
as such but what I found best were Hashmaps.

Hashmaps allow you to have a key and a value which fits perfectly
with this project. You cannot have duplicates in a hashmap and you
can easily check for a value with the get method it contains.

Project

The backend development of the project was fairly quick due

longer to create because it was my first time attempting such.

If I were given more time I would love to have add rotating images

On a scale from 1-10 I would scale this project an 10 because it was
a lot of fun making and working on.
